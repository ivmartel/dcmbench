About

This is a collection of tools for DICOM javascript parsers. Check out the source repository on github: dcmbench. Apart from ctk-dcmjs (which does not have an associated npm package), all parsers should be of their latest version.

Parser list:

  • ctk-dcmjs: c++ parser (dcmtk) compiled to js
  • daikon: js parser of the rii-mango project
  • dicomParser: js parser of the cornerstonejs project
  • dcmjs: js parser of the dcmjs-org project
  • dwv: js parser by ivmartel
